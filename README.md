# Gender Pay Equality Analysis

## Project Overview
This project analyzes employee compensation data across different factories and job roles to evaluate gender pay equality.

## Objective
To classify Equality Scores into three categories:
- Fair
- Unfair
- Highly Discriminative

## Tools Used
- Microsoft Excel

## Methodology
- Used Excel formulas to analyze Equality Score
- Applied conditions to categorize data into equality classes

## Outcome
The analysis helps identify inequality patterns in salary distribution across roles and locations.
